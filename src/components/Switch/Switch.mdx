# Switch

A toggle switch built on Radix Switch for **instant on/off settings** (e.g. notifications, dark mode,
feature flags). Matches the Swiss Army Figma: 36Ã—20 pill, 12px thumb, off = grey thumb on a white
track, on = white thumb on a primary track. Pure CSS, semantic tokens, RTL-aware.

## When to use

- **Instant settings** â€” toggles that apply immediately (notifications, dark mode, WiFi on/off).
- **Not form submission** â€” Switch changes take effect at once; use **Checkbox** for selections
  that are submitted with a form.
- **Boolean states only** â€” for three or more options, use Radio or Select.
- **vs Checkbox** â€” both are binary, but Switch implies "this setting is live now"; Checkbox is
  part of a form that you'll submit later.

## Off and On

The switch is **uncontrolled by default** via `defaultChecked`, or **controlled** with `checked`
and `onCheckedChange`:

<Canvas of={SwitchStories.Off} />

<Canvas of={SwitchStories.On} />

```tsx
// Uncontrolled
<Switch defaultChecked={true} />

// Controlled
const [isOn, setIsOn] = useState(false);
<Switch checked={isOn} onCheckedChange={setIsOn} />
```

## Label

Pass `label` to render text next to the toggle (inline-start in LTR, inline-end in RTL):

The label is wired via `htmlFor`/`id` automatically, so clicking it toggles the switch.

## Disabled vs Read-Only

Both states make the switch inert (can't toggle), but differ in appearance:

<Canvas of={SwitchStories.Disabled} />

<Canvas of={SwitchStories.ReadOnly} />

- **Disabled** (`disabled`) â€” faded (50% opacity), removed from tab order. Use when the setting
  doesn't apply.
- **Read-only** (`readOnly`) â€” full colour, removed from tab order. Use when you want to **show**
  a value the user can't change (e.g. a system-controlled setting).

Both are implemented via Radix's `disabled` under the hood â€” they're genuinely non-interactive
and announced correctly to assistive tech.

## RTL

The thumb travels via logical CSS (`inset-inline-start`), so in RTL the off thumb sits on the
right and slides left when on. No extra configuration needed.

## Accessibility

- **Semantics** â€” Radix renders `role="switch"` and `aria-checked` (true/false/mixed).
- **Keyboard** â€” `Space` or `Enter` toggles the switch; focus is visible.
- **Label** â€” passing `label` wires `htmlFor`/`id` (or pass your own `id`); clicking the label
  toggles the switch. Without a label, provide `aria-label`.
- **Read-only** â€” sets `aria-readonly` and removes from tab order.

## Token-driven styling

Every colour, size, border, and motion value is a semantic token â€” no hardcoded values. The
track is 36Ã—20 with `--radius-full`, the thumb is 12px, and the on-track colour is
`--color-background-selected-default`, off-thumb is `--color-icon-placeholder`, on-thumb is
`--color-icon-oncolor-default`. Themes switch with no component changes.
